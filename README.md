# Minesweeper – лабораторні роботи з веб-дизайну (Абакумов Антон КС-32)

Репозиторій містить послідовну реалізацію гри **«Мінер»** в межах 4 лабораторних робіт:

- верстка інтерфейсу (HTML + CSS);
- побудова структури даних у JavaScript;
- базова логіка гри;
- інтеграція логіки з DOM та подіями користувача.

---

## Структура проєкту

```text
Abakumov-Anton/
  lab1/
    Minesweeper.html     ← ЛР1: верстка інтерфейсу (тільки HTML+CSS)
  lab2/
    Minesweeper2.html    ← ЛР2: структура даних гри (масиви, об’єкти, стан гри)
  lab3/
    Minesweeper3.html    ← ЛР3: логіка гри (generateField, openCell, toggleFlag, таймер)
  lab4/
    Minesweeper4.html    ← ЛР4: інтерактивна версія з DOM, кліками та таймером
Опис лабораторних робіт
Лабораторна робота 1 – Інтерфейс гри

Верстка основного екрана гри «Мінер».

Реалізовано:

хедер з таймером, лічильником прапорців та кнопкою «Нова гра»;

поле з клітинками;

візуальні стани клітинок (закрита, відкрита, міна, прапорець тощо).

Файл: Abakumov-Anton/lab1/Minesweeper.html

Лабораторна робота 2 – Структура даних гри

Описано структуру клітинки:

yeMina, kilkistSusidnihMin, isOpened, isFlagged (або аналогічний стан).

Побудовано двовимірний масив поля.

Описано об’єкт стану гри (stanHry).

Ініціалізовано тестове поле та вивід у консоль.

Файл: Abakumov-Anton/lab2/Minesweeper2.html

Лабораторна робота 3 – Логіка гри

Реалізовано функції:

generateField(rows, cols, mines) – генерація випадкового поля;

countNeighbourMines(field, row, col) – підрахунок мін навколо клітинки;

openCell(row, col) – відкриття клітинки, рекурсивне відкриття нулів, поразка;

toggleFlag(row, col) – встановлення / зняття прапорця;

startTimer() / stopTimer() – таймер на setInterval.

Перевірка логіки через консоль браузера.

Файл: Abakumov-Anton/lab3/Minesweeper3.html

Лабораторна робота 4 – Інтерактивний інтерфейс (DOM)

Інтегровано логіку гри з DOM:

рендеринг поля на основі даних;

обробка подій:

ліва кнопка миші – відкриття клітинки;

права кнопка (contextmenu) – встановлення / зняття прапорця;

динамічний таймер;

лічильник доступних прапорців;

кнопка «Старт / Рестарт».

Реалізовано логіку завершення гри (перемога / поразка) та відображення мін.

Файл: Abakumov-Anton/lab4/Minesweeper4.html
